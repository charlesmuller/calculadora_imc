# calculadora_imc
 
## Criado em PHP por Charles Müller da Silva ##

- Para executar em windows:
1 - Baixar xampp (https://www.apachefriends.org/pt_br/download.html);
2 - No xampp ativar o apache e o mysql;
3 - Inserir os arquivos clonados do github dentro da pasta htdocs que está dentro da pasta onde o xampp foi instalado no pc (geralmente no disco C:);
4 - Abrir o navegador em localhost/pasta-onde-estão-os-arquivos.

- Para executar em linux:
1 - Clonar o repositório do git;
2 - Rodar no terminal o comando php -S localhost:2020 (ou a porta que preferir) dentro do diretório onde foi clonado a aplicação;
3 - Abrir localhost/pasta-onde-estão-os-arquivos no navegador.
